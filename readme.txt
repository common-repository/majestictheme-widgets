== Description ==

This plugin is a collection of 4 widgets, specially made for free themes by MajesticTheme. But these widgets are not
strictly coupled with the themes by MajesticTheme. Anyone can use these widgets with any theme.

The 4 widgets are:
1. MTW: Blurb
2. MTW: Clients
3. MTW: Member
4. MTW: Testimonials
5. MTW: Skillbar [New - 1.1.0]


== Frequently Asked Questions ==

= Can I use these widgets with any theme? =

Yes. You can use all these widgets with any theme. And if you use with themes from MajesticTheme then you'll get some extra styling.
